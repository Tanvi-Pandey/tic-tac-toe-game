# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

- Two-player gameplay
- Winner detection
- Draw detection
- Reset Game button
- New Game button
- Responsive design
- Interactive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Game Rules

* Player O starts first
* Players take turns placing marks
* First player to align 3 symbols wins
* If all boxes are filled and no winner is found, the game ends in a draw
